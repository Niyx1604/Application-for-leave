## 🎯 USER WORKFLOW

1. **Launch Application**
   ```bash
   python leave_app_refactored.py
   ```

2. **Configure Logos** (Optional)
   - Click "Browse PhilHealth Logo" to select logo file
   - Click "Browse Bagong Pilipinas Logo" to select logo file
   - Or place files in `logos/` directory

3. **Fill Form**
   - **Name**: Enter Last, First, Middle names separately
   - **Position**: Select from dropdown (all Civil Service positions)
   - **Date Filing**: Auto-filled with current date
   - **Leave Type**: Check appropriate boxes (13 types available)
   - **Inclusive Dates**: Use calendar widgets
   - **Working Days**: Select from dropdown
   - **Commutation**: Choose requested/not requested

4. **Generate PDF**
   - Click "Generate PDF" button
   - Professional PDF created with grid-based layout
   - File saved as `Application_for_Leave.pdf`

5. **Result**
   - Print-ready A4 format
   - Professional appearance
   - All data properly positioned
   - Vector checkboxes and integrated logos
